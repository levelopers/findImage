# find image url links in files

### support file types


-  .html
-  .wxml
-  .css
-  .vue
-  .js

### instructions

#### node.js

open images in browser: 
> `node index.js open`

print link in command tool:  
> `node index.js`

#### html

run  `node server.js` open browser on  `http://localhost:8080/`

(double click on image open in new tab)
